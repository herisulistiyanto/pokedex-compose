## PokeDex Jetpack Compose

### A Jetpack Compose demo
Just some Jetpack Compose playground demo which fetch data from network and display it on the 
screen.

### Tech Stack 
Library used in this project are :
* Jetpack Compose
* Hilt
* MVVM
* Retrofit 2
* Moshi
* Coil Accompanist
* Coroutine
* Navigation Component
